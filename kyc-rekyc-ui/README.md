# Agentic AI-Based KYC / Re-KYC Automation Platform UI

Complete React + Node source code for a Housing Finance KYC/Re-KYC platform prototype.

## Modules Included
- CXO / Operations Command Center
- Customer Onboarding Studio
- KYC Case Management
- Agentic AI Hub
- Workflow Automation Engine
- AML Risk & Fraud Intelligence
- Compliance & Audit Console
- Admin, RBAC & Integration Configuration

## Tech Stack
- Frontend: React, Vite, React Router, Recharts, Lucide Icons
- Backend: Node.js, Express mock API

## Run Locally
```bash
npm run install:all
npm run dev
```
Frontend: http://localhost:5173
Backend API: http://localhost:8080/api/health

## Production Build
```bash
npm run build
npm start
```

## Notes
This is a UI/UX prototype with mock APIs. Integrate real Aadhaar, PAN, CKYC, DigiLocker, OCR, face match, liveness, AML and DMS services through the server API layer.
