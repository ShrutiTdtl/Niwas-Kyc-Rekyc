📄 Scope of Work (SoW)
Agentic AI-Based KYC / Re-KYC Automation Platform for Housing Finance

1. 🎯 Objective
Design and implement an AI-first, agentic KYC platform that:
Automates end-to-end KYC & Re-KYC lifecycle 
Ensures 100% compliance with RBI / NHB / PMLA / CKYC norms 
Supports multi-source, multi-channel customer onboarding 
Enables real-time validation, fraud detection, and risk scoring 
Reduces manual intervention by >80% 
Provides audit-ready, regulator-compliant traceability 

2. 🧠 Solution Overview (Agentic AI Architecture)
🔷 Transformation Vision
From:
Document-based KYC → Digital KYC → Autonomous Agentic KYC Intelligence Platform

3. 🤖 Agentic AI Multi-Agent Framework
🧠 Core Agents

1. 📥 Customer Intake Agent
Sources: 
Direct branch onboarding 
Digital onboarding (web/mobile apps) 
DSAs / Channel partners 
Builders / brokers 
Co-lending partners 
Captures: 
Personal details 
Property details 
Financial profile 

2. 📄 Document Intake & OCR Agent
Accepts: 
Aadhaar, PAN, Passport, Driving License, Voter ID 
Utility bills, bank statements, salary slips 
Capabilities: 
OCR + AI-based extraction 
Document classification 
Multi-language parsing 

3. 🧾 Document Understanding Agent
Extracts: 
Name, DOB, Address 
Document numbers 
Validates: 
Cross-document consistency 
Generates: 
Structured KYC profile 

4. 🔍 Identity Verification Agent
Integrates: 
Aadhaar eKYC (Offline XML / OTP) 
PAN verification APIs 
CKYC registry 
DigiLocker 
Performs: 
Face match (selfie vs document) 
Liveness detection 

5. 🏠 Address Verification Agent
Verifies: 
Current & permanent address 
Methods: 
Document validation 
Geo-tagging 
Field verification triggers 

6. 🧠 Risk Scoring & AML Agent
Evaluates: 
Customer risk profile (Low/Medium/High) 
Checks: 
Sanctions lists 
PEP lists 
Adverse media 
Assigns: 
AML risk score 

7. 🔁 Re-KYC & Periodic Review Agent
Triggers: 
Periodic KYC updates (as per risk category) 
Tracks: 
Expiry of documents 
Change in customer profile 

8. ⚠️ Fraud Detection Agent
Detects: 
Fake documents 
Identity duplication 
Suspicious patterns 
Uses: 
AI anomaly detection 

9. 📊 Compliance & Audit Agent
Maintains: 
Complete audit trail 
Generates: 
Regulatory reports 
KYC compliance dashboards 

10. 📣 Communication Agent
Sends: 
KYC reminders 
Re-KYC alerts 
Document deficiency notifications
